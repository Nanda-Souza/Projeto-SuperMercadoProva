***Projeto Super Mercado***

**Descrição:**

O projeto SuperMercadoProva simula o sistema de gerenciamento de vendas de um supermercado, com controle de estoque, verificação da existência produtos escolhidos no pedido e demais funcionalidades

**Funcionalidades:**

**Fluxograma**

**Tecnologias Utilizadas*
Java <br>
Programação Orientada a Objetos (POO)
